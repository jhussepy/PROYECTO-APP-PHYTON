# Ethical Use

PySec Academy Elite se orienta a formación, defensa, hardening y análisis autorizado.

## Contextos permitidos

- sistemas propios
- `localhost` y `127.0.0.1`
- máquinas virtuales y contenedores propios
- CTF y plataformas educativas
- sistemas con autorización explícita y alcance documentado

## Contextos prohibidos

- terceros sin permiso
- robo o prueba de credenciales reales
- phishing
- malware
- persistencia o evasión
- exfiltración
- automatización ofensiva no autorizada
- destrucción o alteración de evidencia

## Protocolo de laboratorio

1. Confirma propiedad o autorización.
2. Define alcance, fecha y límites.
3. Usa datos ficticios.
4. Registra evidencia mínima.
5. Prioriza mitigación.
6. Ejecuta re-test.
7. Elimina datos de laboratorio cuando corresponda.

Juice Shop y WebGoat deben ejecutarse localmente o en una plataforma que autorice expresamente la práctica.
